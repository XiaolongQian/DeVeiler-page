# DeVeiler Project Page

This repository hosts the static site for **“Learning Latent Transmission and Glare Maps for Lens Veiling Glare Removal.”**  
The live page summarizes VeilGen (our degradation generator) and DeVeiler (our restoration model), and showcases qualitative comparisons on the Realworld-Compound datasets.
